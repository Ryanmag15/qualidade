Instruções

Para iniciar o aplicativo, siga estas etapas:

1. Abra o terminal. 2. Navegue até o diretório frontend/admin com o comando: cd frontend/admin 3. Execute o comando para iniciar o servidor de desenvolvimento: npm run dev

Acesse o aplicativo no seguinte endereço:

Local: http://localhost:5173/

Network: http://10.255.255.254:5173/

Network: http://172.22.222.102:5173/
